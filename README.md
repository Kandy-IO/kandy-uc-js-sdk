# @kandy-io/uc-sdk

## Install

### Using npm :

`$ npm install @kandy-io/uc-sdk`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-uc-js-sdk#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/uc-sdk@4.13.0/dist/kandy.js"
  integrity="sha384-nHNi1tRSz7q2CPIFs53Zl+d1fmF4B9KjzAGM91mzr5ucdhaY1Bvy3asYfqsfpIvz"
  crossorigin="anonymous">
</script>
```
